# Windsanity Mini-Splits Website

A modern, lightweight contractor website built to replace a Squarespace site with a faster, cheaper, and more customizable solution.

The project is designed to serve two purposes:

1. A production website for Windsanity Mini-Splits (HVAC contractor)
2. A reusable template for building future small business websites

---

# Tech Stack

- Vanilla JavaScript
- Vite
- Netlify hosting
- Netlify Forms
- Component-based architecture
- CSS design system using custom properties

This project intentionally avoids large frameworks to keep the site fast, simple, and easy to maintain.

---

# Project Goals

- Improve performance over Squarespace
- Reduce hosting costs
- Enable full customization
- Create a reusable template for other service businesses
- Maintain simple infrastructure
