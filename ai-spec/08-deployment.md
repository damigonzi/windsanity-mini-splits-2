# Deployment

## Netlify Deployment
- Site is deployed to Netlify
- Vite build output is published to Netlify
- Netlify Forms are enabled for form handling
- Netlify email and Slack notifications are configured

## Build/Publish
- Run `vite build` for production
- Publish the `dist/` folder to Netlify

## Domain & Notifications
- Custom domain setup is assumed
- Notification integrations are ready for future enhancements

## Reference Inputs
- Use Netlify and Vite documentation for detailed setup
