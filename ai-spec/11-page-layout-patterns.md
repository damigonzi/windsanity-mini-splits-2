# 11-page-layout-patterns.md

This document defines **standard page layout patterns** used throughout
the Windsanity Mini-Splits website.

These patterns guide AI-assisted generation to ensure: - consistent page
structure - predictable conversion flow - reusable section composition -
better user experience - strong lead generation

Each page should follow a **structured content flow** rather than
arbitrary section ordering.

------------------------------------------------------------------------

# Global Page Layout Structure

All pages should follow this general structure:

1.  Header / Navigation
2.  Hero Section
3.  Primary Content Sections
4.  Supporting Content Sections
5.  Call-to-Action Section
6.  Contact Prompt (optional)
7.  Footer

Example layout:

Header\
Hero\
Value Proposition\
Primary Content\
Supporting Content\
CTA Banner\
Footer

------------------------------------------------------------------------

# Header Pattern

The header should include:

-   Logo (left)
-   Navigation links (center or right)
-   CTA button (optional)

Example navigation:

-   Home
-   Services
-   Projects
-   Calculator
-   Blog
-   Contact

Primary CTA example: **Request Estimate**

Header should be sticky or persistent during scrolling when practical.

------------------------------------------------------------------------

# Hero Section Pattern

The hero section is the **most important conversion area** on the page.

Hero structure:

-   Headline
-   Subheadline
-   Short description
-   Primary CTA
-   Optional secondary CTA
-   Optional image

Example:

Headline\
Efficient Mini-Split Installation for Homes and Businesses

Subheadline\
Professional ductless HVAC solutions designed for comfort, efficiency,
and long-term reliability.

Primary CTA\
Request a Free Estimate

Secondary CTA\
Explore Services

Hero layout may be: - text left + image right - centered stacked
layout - full-width hero banner

------------------------------------------------------------------------

# Value Proposition Section

This section explains **why the service matters**.

Typical layout:

-   Section Title
-   Short explanation
-   3--4 benefit cards

Example benefits:

-   Energy Efficient
-   Quiet Operation
-   Zoned Temperature Control
-   Flexible Installation

------------------------------------------------------------------------

# Services Page Pattern

Recommended order:

Hero\
Services Overview\
Service Cards Grid\
Project Examples\
CTA Banner\
Contact Section

Service cards should include:

-   Icon or image
-   Title
-   Short description
-   CTA button

Typical grid layout:

Desktop: 3 cards per row\
Mobile: 1 card per row

------------------------------------------------------------------------

# Projects / Gallery Page Pattern

Recommended structure:

Hero\
Intro text\
Project grid\
CTA Banner\
Contact prompt

Project card example:

-   Image
-   Project title
-   Short description
-   Optional location

Images may be **hard-coded initially** but structured so they can later
become data-driven.

------------------------------------------------------------------------

# Cost Calculator Page Pattern

Recommended structure:

Hero\
Calculator explanation\
Calculator widget\
Supporting explanation\
CTA Banner\
Contact prompt

The calculator should be treated as:

-   an interactive widget
-   embedded within the page
-   not a full SPA application

Calculator section should include:

-   Input fields
-   Calculate button
-   Estimated result output
-   Optional contact prompt

------------------------------------------------------------------------

Hero\

Blog card format:

-   Image
-   Title
-   Short excerpt

# About Page Pattern

Recommended order:

Hero\
Company story\
Experience highlights\
Team or leadership section\

-   veteran-owned leadership
-   industry experience
-   reliability
-   customer service

------------------------------------------------------------------------

# Contact Page Pattern

Recommended order:

Hero\
Contact description\
Contact form\
Additional contact info\
CTA reassurance

Form fields:

-   Name
-   Email
-   Phone
-   ZIP Code
-   Service Type
-   Message

After submission:

-   show success message
-   optionally redirect to a thank-you state

------------------------------------------------------------------------

# CTA Banner Pattern

CTA sections should appear multiple times across pages.

Structure:

-   Headline
-   Short supporting text
-   Primary CTA button

Example:

Headline: Upgrade Your Comfort with a Mini-Split System\
CTA: Request a Free Estimate

CTA sections should be visually distinct from surrounding sections.

------------------------------------------------------------------------

# Footer Pattern

Footer should include:

-   Business contact info
-   Navigation links
-   Optional social links
-   Copyright notice

Example links:

-   Home
-   Services
-   Projects
-   Blog
-   Calculator
-   Contact

Example contact info:

Email: windsanity.mini.split@gmail.com\
Phone: (402) 613-5142

------------------------------------------------------------------------

# Page Composition Rules

When generating pages:

-   Always start with a Hero section.
-   Do not stack multiple CTA banners directly next to each other.
-   Alternate visual layouts when possible (image/text variation).
-   Keep sections focused and scannable.
-   Use cards or grids for repeated content.

------------------------------------------------------------------------

# Conversion Strategy

Every page should include at least **one strong CTA**.

Preferred CTAs:

-   Request a Free Estimate
-   Schedule Service
-   Start Cost Estimate
-   Contact Us

CTAs should appear:

-   in the hero
-   mid-page
-   near the end of the page

------------------------------------------------------------------------

# Mobile Layout Guidance

Pages should be **mobile-first**.

Guidelines:

-   Stack columns vertically
-   Avoid overly large hero heights
-   Ensure buttons are easily tappable
-   Maintain readable line lengths

------------------------------------------------------------------------

# Future Template Reuse

Page layouts should be designed so they can be reused for other service
businesses.

Reusable section types:

-   hero
-   service cards
-   testimonial cards
-   project gallery
-   CTA banner
-   contact form
